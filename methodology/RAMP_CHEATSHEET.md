# RAMP Quick Reference

Learn RAMP in 60 seconds.

---

## The One Question

> **"Can we undo this?"**

---

## The Five Levels

| Level | Time to Undo | Action |
|-------|--------------|--------|
| ⚡ L1 | <1 hour | Ship it |
| 🔄 L2 | <1 day | Monitor |
| 📋 L3 | <1 week | Plan first |
| ⚠️ L4 | <1 month | Approval |
| 🚫 L5 | Never | Full stop |

---

## The Direction

- **🟢 RAMP DOWN** (L1-L2): Move fast
- **🟡 RAMP CAREFULLY** (L3): Add checkpoints
- **🔴 RAMP UP** (L4-L5): Add rigor

---

## The Protocol (RAMPKIT)

```
R, Recognize ("What?")
A, Assess ("How reversible?")
M, Map ("What if it fails?")
P, Position ("Up or down?")
K, Kickoff ("Who owns it?")
I, Inspect ("Check signals")
T, Track ("What did we learn?")
```

---

## The Rapid Version (MVR)

**15 minutes**: R → A → M → P

---

## The Principles (DOORS)

```
D, Declare the door type
O, Observe irreversibility triggers
O, Own the rollback path
R, Ready the rollback before shipping
S, Signal when to reverse
```
